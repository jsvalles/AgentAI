Chatbox Sencillo

Instrucciones rápidas:

- Abre `chatbox/index.html` en tu navegador (doble clic o "Abrir con...").
- Escribe en el campo de entrada y pulsa `Enviar` o Enter.
- El bot responde con respuestas simuladas (saludos, gracias, adiós o repite el texto).

Notas:

- Es una aplicación cliente estática; no necesita servidor.
- Si deseas conectar con una API, modifica `script.js` para enviar requests a tu backend.
